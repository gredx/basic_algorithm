# basic_algorithm
包含了一些典型算法，数据结构的实现，有的函数参考了cppreference.

| 文件            | 功能           |
| --------------- | -------------- |
| items           | 算法的实现     |
| test            | 测试算法的代码 |
| utility         | 一些工具函数   |
| basic_algorithm | 主函数         |

| item中的算法  | 简介                       |
| ------------- | -------------------------- |
| binary_search | 二分查找                   |
| lower_bound   | 找下界                     |
| make_heap     | 建堆,参考c++11 algorithm库 |
| push_down     | 对堆的调整                 |
| pop_heap      | 弹出堆顶元素               |
